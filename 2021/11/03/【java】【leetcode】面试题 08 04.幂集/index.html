题目链接： 幂集

题目描述：幂集。编写一种方法，返回某集合的所有子集。集合中不包含重复的元素。

说明：解集不能包含重复的子集。

输入： nums = [1,2,3]

输出：

[

[3], [1], [2], [1,2,3], [1,3], [2,3], [1,2], []

]

非递归模拟实现：

假设 nums = [1,2,3]

那么 nums 的幂集包括有：

[],[1],[2],[1,2],[3],[1,3],[2,3],[1,2,3]

我们先向结果集 res 中加入一个空集合，然后对整个 nums 数组进行循环遍历，让数组中的元素加入到子集中，然后变成新的子集。

然后遍历子集，将子集封装成新的子集。在子集后继续添加元素，然后将子集添加到 res 结果集中即可。
